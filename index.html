TileSim

TileSim app was created as a proof of concept for a client of Chervic Advisory Services.

Tilesim provides an augmented preview of the ground and wall tiles of a building which can help customers make an informed choice of the tiles to be laid in their homes with the approximate tile count and cost estimate.

App Development Challenges.
  • Using a Line Renderer to define the boundaries of detected planes for more precise and cleaner rendering of the tiles.
  • Tiles are generated based on the tile size selected by the user.
Role : AR/C# Developer | UI/UX Designer | Sound Designer
  • Coding Language: C#
  • Game Engine: Unity 3D
  • SDK : ARFoundation and Vuforia
